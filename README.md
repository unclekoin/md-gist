# Github Gist
1. Install extension for VSCode `Gist` author Ken Howard
2. Press `F1`
3. Enter `gist`
4. For first enter:
- get token on Github 
- select `Select Profile`
5. Select `Create New Gist`
6. Enter description
7. Private? Y or N
8. Get code from Gist:
- `Open Gist`
- `Insert text from Gist file`
